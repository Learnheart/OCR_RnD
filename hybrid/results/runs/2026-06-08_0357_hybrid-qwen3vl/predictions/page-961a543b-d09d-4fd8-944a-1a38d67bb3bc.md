# 3. 积分定理

$F(s) = L[f(t)] \Rightarrow L[\int_{0}^{t} f(t)dt] = \frac{F(s)}{s} + \frac{1}{s}f^{(-1)}(0)$

其中 $f^{(-1)}(0) = \int_{0}^{t} f(t)dt$

$L[\int_{0}^{t} \cdots \int_{0}^{t} f(t)dt \cdots dt] = \frac{F(s)}{s^n} + \frac{1}{s}f^{(-1)}(0) + \cdots + \frac{1}{s}f^{(-n)}(0)$

其中 $f^{(-n)}(0) = \int_{0}^{t} \cdots \int_{0}^{t} f(t)dt \cdots dt$

# 4. 初值定理（重点掌握）

$f(t), \dot{f}(t)$ 可拉氏变换，且 $\lim_{s \to \infty} sF(s)$ 存在

$\Rightarrow \lim_{t \to 0} f(t) = \lim_{s \to \infty} sF(s)$
