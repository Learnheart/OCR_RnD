Table 5.8 End point Constraints for a Cubic Spline

| Description of the strategy | Equations involving $m_1$ and $m_2$ |
| :--- | :--- |
| (a) Clamped cubic spline: specify $S'(x_0), S'(x_n)$ (the “best choice” if the accelerations are known) | $m_1 = \frac{1}{h} \sum_{j=1}^{n} (S'(x_j) - \frac{m_2}{2})$<br>$m_2 = \frac{1}{h} \sum_{j=1}^{n} (S'(x_j) - \frac{m_1}{2})$<br>$\frac{m_1 + m_2}{2}$ |
| (b) Natural cubic spline (a “relaxed curve”) | $m_1 = 0, m_2 = 0$ |
| (c) Extrapolate $S'(x)$ to the endpoints | $m_1 = m_2$<br>$m_1 = m_2 + \frac{h}{2} \sum_{j=1}^{n} (m_2 - m_1)$<br>$\frac{m_1 + m_2}{2}$ |
| (d) $S''(x)$ is constant near the endpoints | $m_1 = m_2, m_1 = m_2 + \frac{h}{2} \sum_{j=1}^{n} (m_2 - m_1)$<br>$\frac{m_1 + m_2}{2}$ |
| (e) Specify $S'(x)$ at each endpoint | $m_1 = S'(x_0), m_2 = S'(x_n)$ |
